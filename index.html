大地工程國考公式速記網

v6.0 最終校對版

上篇：土壤力學

三相圖基本定義

$e = \frac{V_v}{V_s}$    $w = \frac{W_w}{W_s}$
$S = \frac{V_w}{V_v}$    $n = \frac{V_v}{V_t}$

老師劃重點：

孔隙比(e)和孔隙率(n)是兄弟，分母一個是土粒體積(Vs)，一個是總體積(Vt)。考試最愛考它們的互換：$n = \frac{e}{1+e}$ 或 $e = \frac{n}{1-n}$，必背！

單位重家族

$\gamma_m = \frac{G_s + Se}{1+e}\gamma_w = \gamma_d(1+w)$
$\gamma_{sat} = \frac{G_s + e}{1+e}\gamma_w$
$\gamma' = \gamma_{sat} - \gamma_w = \frac{G_s - 1}{1+e}\gamma_w$
$\gamma_s = \gamma_d(1+e) = G_s \gamma_w$

老師劃重點：

所有單位重公式的核心都是 $\frac{G_s + Se}{1+e}\gamma_w$。乾土 S=0，飽和土 S=1。$\gamma_s$是土粒單位重，不要跟比重$G_s$搞混！

萬用公式 (天王公式)

$Se = wG_s$

老師劃重點：

這條公式一定要刻在DNA裡！速記口訣：「騷吾妻」(S w Gs)。它是所有三相圖計算的連結橋樑，解題卡關時，想想「騷吾妻」就對了！

相對密度 $D_r$

$D_r = \frac{e_{max} - e}{e_{max} - e_{min}}$
$= \frac{\gamma_{d,max}(\gamma_d - \gamma_{d,min})}{\gamma_d(\gamma_{d,max} - \gamma_{d,min})}$

老師劃重點：

相對密度是判斷「砂土」鬆密程度的指標。除了用孔隙比e，也可以用乾單位重$\gamma_d$來算，兩個公式都要記。注意，這是砂土專用！

阿太堡限度

$PI = LL - PL$
$LI = \frac{w_N - PL}{PI}$

老師劃重點：

PI代表土壤塑性範圍。LI > 1 土壤呈液態，LI < 0 呈固態。這是判斷黏土現地狀態的重要指標。

A-line & U-line

A-line: $PI = 0.73(LL-20)$
U-line: $PI = 0.9(LL-8)$

老師劃重點：

A-line區分黏土(C)和粉土(M)。U-line是經驗上限，真實土壤數據點應在U-line下方，若在上，表示實驗數據有誤！

粒徑分佈參數

均勻係數: $C_u = \frac{D_{60}}{D_{10}}$
曲率係數: $C_c = \frac{(D_{30})^2}{D_{10} \times D_{60}}$

老師劃重點：

用來判斷粗粒土級配。級配良好砂土(SW): $C_u \ge 6$, $1 \le C_c \le 3$。級配良好礫石(GW): $C_u \ge 4$, $1 \le C_c \le 3$。

活性 $A_c$

$A_c = \frac{PI(\%)}{\text{黏粒含量}(\%)}$

老師劃重點：

評估黏土礦物膨脹潛能。高活性黏土($A_c > 1.25$)吸水易膨脹，是高嶺石、伊利石、蒙脫石的考點！

NC黏土沉陷

$S_c = \frac{C_c H_0}{1+e_0}\log(\frac{\sigma'_{v0}+\Delta\sigma'}{\sigma'_{v0}})$
經驗公式: $C_c = 0.009(LL-10)$

老師劃重點：

NC黏土沉陷走e-log p'圖的原始壓縮曲線。應力要用「土層中央」點的有效應力來算。

OC黏土沉陷

Case 1: $\sigma'_{f} \le \sigma'_{p}$
$S_c = \frac{C_r H_0}{1+e_0}\log(\frac{\sigma'_{f}}{\sigma'_{v0}})$
Case 2: $\sigma'_{f} > \sigma'_{p}$
$S_c = \frac{H_0}{1+e_0}[C_r\log(\frac{\sigma'_{p}}{\sigma'_{v0}})+C_c\log(\frac{\sigma'_{f}}{\sigma'_{p}})]$

老師劃重點：

OC黏土關鍵是判斷最終應力$\sigma'_f$是否超過預壓密壓力$\sigma'_p$。沒超過走$C_r$；超過了，分兩段算($C_r$+$C_c$)。

壓密速率

時間因數: $T_v = \frac{c_v t}{H_{dr}^2}$
For $U \le 60\%$: $T_v = \frac{\pi}{4}U^2$
For $U > 60\%$: $T_v = 1.781 - 0.933\log(100-U\%)$

老師劃重點：

排水路徑$H_{dr}$是最大陷阱！雙面排水$H_{dr}=H/2$；單面排水$H_{dr}=H$。$T_{50}=0.197, T_{90}=0.848$。

莫爾-庫倫破壞準則

$\tau_f = c' + \sigma'_n \tan\phi'$

老師劃重點：

土壤剪力強度的根本大法！強度由有效凝聚力c'和有效摩擦角$\phi'$組成。

三軸試驗主應力關係

$\sigma'_1 = \sigma'_3 \tan^2(45^\circ+\frac{\phi'}{2}) + 2c'\tan(45^\circ+\frac{\phi'}{2})$

老師劃重點：

這是從莫爾圓幾何關係推導的，必背！$\tan^2(45^\circ+\frac{\phi'}{2})$ 就是被動土壓力係數$K_p$。

不排水剪力強度 $S_u$

$S_u = c_u = \frac{\sigma_1 - \sigma_3}{2} = \frac{q_u}{2}$

老師劃重點：

飽和黏土UU試驗，破壞包絡線是水平線 ($\phi_u = 0$)。無圍壓縮試驗($\sigma_3=0$)是UU試驗的特例。

孔隙水壓參數 (Skempton)

$\Delta u = B[\Delta\sigma_3 + A(\Delta\sigma_1 - \Delta\sigma_3)]$

老師劃重點：

參數A, B用來描述加載引起的超額孔隙水壓。B跟飽和度有關，A跟應力歷史(OCR)有關。

下篇：基礎工程

靜止土壓力係數 $K_0$

NC土: $K_0 = 1 - \sin\phi'$
OC土: $K_{0,oc} = (1 - \sin\phi')OCR^{\sin\phi'}$

老師劃重點：

$K_0$是無側向變形時，水平與垂直有效應力之比。OC土的$K_0$會比NC土大。

c-$\phi$土壓力(通用式)

主動: $\sigma'_a = (\sigma'_v+q) K_a - 2c'\sqrt{K_a}$
被動: $\sigma'_p = (\sigma'_v+q) K_p + 2c'\sqrt{K_p}$
張力裂縫: $z_c = \frac{2c'}{\gamma\sqrt{K_a}}$

老師劃重點：

此為有效應力分析。砂土c'=0，黏土短期分析$\phi_u=0, K_a=K_p=1$可代入簡化。

擋土牆穩定分析

抗滑 $FS_{s} = \frac{\sum R_H}{\sum P_H} \ge 1.5$
抗傾倒 $FS_{o} = \frac{\sum M_R}{\sum M_O} \ge 2.0$
承載力檢核: $q_{max} \le q_a$

老師劃重點：

力矩計算以牆趾(Toe)為支點。承載力檢核要考慮偏心造成之最大壓力。

無限邊坡 FS (c-$\phi$土)

無滲流: $FS = \frac{c'}{\gamma H \cos\beta\sin\beta} + \frac{\tan\phi'}{\tan\beta}$
浸水: $FS = \frac{c'}{\gamma_{sat} H \cos\beta\sin\beta} + \frac{\gamma'}{\gamma_{sat}}\frac{\tan\phi'}{\tan\beta}$
平行滲流: $FS = \frac{c'}{\gamma_{sat} H \cos\beta\sin\beta} + \frac{\gamma'}{\gamma_{sat}}\frac{\tan\phi'}{\tan\beta}$

老師劃重點：

注意公式分母是$\cos\beta\sin\beta$或$\cos^2\beta\tan\beta$，兩者相等。滲流會大幅降低FS。

黏土邊坡 FS ($\phi_u=0$)

$FS = \frac{c_u}{c_d} = \frac{N_s c_u}{\gamma H}$
穩定數: $N_s = \frac{1}{m} = \frac{\gamma H}{c_d}$

老師劃重點：

用於黏土邊坡的圓弧滑動分析。穩定數$N_s$可由Taylor穩定圖查得，是坡角$\beta$和深度因數$n_d$的函數。

通用淨承載力公式

$q_{net,ult} = c'N_c F_{cs}F_{cd} + q(N_q-1)F_{qs}F_{qd} + 0.5\gamma' B' N_\gamma F_{\gamma s}F_{\gamma d}$

老師劃重點：

最常用公式！$q=\gamma D_f$。F是修正因子，s形狀，d深度。$B'$是考慮偏心的有效寬度。

Terzaghi 淨承載力

條形: $q_{net,ult} = c'N_c + q(N_q-1) + 0.5\gamma' B N_\gamma$
方形: $q_{net,ult} = 1.3c'N_c + q(N_q-1) + 0.4\gamma' B N_\gamma$
矩形: $q_{net,ult} = (1+0.3\frac{B}{L})c'N_c + q(N_q-1) + (1-0.2\frac{B}{L})0.5\gamma' B N_\gamma$

老師劃重點：

Terzaghi公式較古老，但觀念重要。注意與通用公式的差異。

容許承載力 $q_a$

$q_a = \frac{q_{net,ult}}{FS} + \gamma D_f$

老師劃重點：

極限承載力除以安全係數(通常是3)。注意淨值和總值的轉換，考試常考。

偏心基礎壓力檢核

當 $e \le B/6$:
$q_{max/min} = \frac{Q}{BL}(1 \pm \frac{6e}{B})$
當 $e > B/6$:
$q_{max} = \frac{4Q}{3L(B-2e)}$

老師劃重點：

用最大壓力$q_{max}$檢核是否小於容許承載力$q_a$。承載力計算則用有效面積法 ($B' = B-2e$)。

黏土中樁承載力

樁尖: $q_p = c_u N_c^* \approx 9c_u$
樁側($\alpha$法): $f_s = \alpha c_u$
樁側($\beta$法): $f_s = \beta \sigma'_{v0} = K \tan\delta \cdot \sigma'_{v0}$
樁側($\lambda$法): $f_s = \lambda(\sigma'_{v0,avg} + 2c_{u,avg})$

老師劃重點：

$\alpha$法(總應力), $\beta$法(有效應力), $\lambda$法(經驗)。

砂土中樁承載力

樁尖: $q_p = q' N_q^* \le q_l$
樁側: $f_s = K \sigma'_{v0} \tan\delta \le f_l$

老師劃重點：

$q'$是樁尖有效覆土壓力。樁尖阻力和側摩擦力都有一個極限值($q_l, f_l$)，不會隨深度無限增加，臨界深度$L_c \approx 15B \sim 20B$。

樁群承載力

效率法: $Q_{ug} = E (\sum Q_u)$
塊狀破壞(黏土):
$Q_{ug} = c_u N_c^* A_g + P_g D_f \bar{c_u}$

老師劃重點：

樁群承載力取[效率法]和[塊狀破壞]的較小值。$A_g$是樁群底面積，$P_g$是樁群周長。

樁群效率 E

Converse-Labarre 公式:
$E = 1 - \frac{\theta}{90} \frac{(n-1)m+(m-1)n}{mn}$
$\theta = \arctan(d/s)$

老師劃重點：

m, n是樁的行數與列數，d是樁徑，s是樁心距。樁距越近，效率越低。

Peck 經驗土壓力圖

砂土: 梯形, $p_{max}=0.65K_a\gamma H$
軟~中等黏土: 梯形, $p_{max}=\gamma H-4S_u$ or $0.3\gamma H$ (取大值)
硬黏土: 梯形, $p_{max}=0.2\gamma H \sim 0.4\gamma H$

老師劃重點：

這是用於支撐系統設計的「視土壓力圖」，與Rankine理論不同，分佈形狀與大小是經驗值，必背！

開挖抗隆起分析

Terzaghi: $FS = \frac{5.7c_u}{q+\gamma H - c_u H/B} \ge 1.5$
Eide & Bjerrum: $FS = \frac{N_c c_u}{q+\gamma H} \ge 1.2$
正規抗隆起: $FS = \frac{M_R}{M_d} \ge 1.2$

老師劃重點：

用於評估軟弱黏土層開挖底部的穩定性。q是開挖面上的荷重，$N_c$可查圖。

貫入深度檢核

$FS = \frac{P_p + M_s}{P_A} \ge 1.5$

老師劃重點：

此為擋土壁(如鋼版樁)的穩定檢核，抵抗力來自牆前的被動土壓力($P_p$)與錨拉力($M_s$)，驅動力為主動土壓力($P_A$)

工地調查

面積比 $A_r = \frac{D_o^2-D_i^2}{D_i^2} \le 10\%$
內淨空比 $C_i = \frac{D_s-D_e}{D_e} = 1 \sim 1.5\%$
岩心回收率 $CR = \frac{\text{回收岩心長}}{\text{理論鑽取長}}$
岩石品質指標 $RQD = \frac{\sum \text{大於10cm岩心長}}{\text{理論鑽取長}}$

老師劃重點：

$A_r$越小，土樣擾動越小。RQD是評估岩盤品質的重要指標。

夯實工程

相對夯實度 $R.C. = \frac{\gamma_{d,field}}{\gamma_{d,max,lab}} \times 100\%$
零孔隙曲線: $\gamma_{zav} = \frac{G_s \gamma_w}{1+wG_s}$

老師劃重點：

R.C.是工地現場乾單位重與實驗室最大乾單位重的比值，是夯實品質的驗收標準。

液化分析

$FS_L = \frac{CRR}{CSR} \ge 1.0$
$CSR = \frac{\tau_{avg}}{\sigma'_{v0}} = 0.65 \frac{a_{max}}{g} \frac{\sigma_{v0}}{\sigma'_{v0}} r_d$

老師劃重點：

液化潛能評估。CSR是地震引致的應力比，CRR是土壤的抗液化應力比(可由SPT-N值等求得)。

樁載重試驗 (Davisson)

破壞載重 $Q_u$ 對應之沉陷:
$S_u(mm) = 3.81 + \frac{B(mm)}{120} + \frac{Q_u L}{A_p E_p}$

老師劃重點：

此為經驗判別法，在載重-沉陷曲線上，找出符合此沉陷量的點，對應的載重即為$Q_u$。
